## 🚀 Como Executar

### Passo 1: Instalar Dependências

```bash
# Criar e ativar virtual environment
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows

# Instalar dependências
pip install -r requirements.txt
```

### Passo 2: Configurar `.env`

```bash
# Crie um arquivo .env na raiz do projeto
OPENAI_API_KEY=sk-sua-chave-aqui
```

### Passo 3: Executar

```bash
python app.py
```

## ⚙️ Configurações e Customizações

### Alterar modelo LLM

```python
# Em app.py, linha ~125
llm = ChatOpenAI(
    model="gpt-4-turbo",  # ← Mudar aqui
    temperature=0,
)
```

### Alterar tamanho de chunks

```python
# Em app.py, função carregar_base_conhecimento()
text_splitter = RecursiveCharacterTextSplitter(
    chunk_size=1000,       # ← Aumentar para documentos maiores
    chunk_overlap=200,
)
```

### Alterar número de documentos recuperados

```python
# Em app.py, função carregar_base_conhecimento()
return vectorstore.as_retriever(search_kwargs={"k": 5})  # ← Top-5 ao invés de Top-3
```

## 🐛 Troubleshooting

### ❌ Erro: "No module named 'faiss'"
```bash
pip install faiss-cpu
```

### ❌ Erro: "duplicate table 'message_store'"
Significa que o banco foi inicializado. É normal após primeira execução.

### ❌ Erro: "knowledge_base.md não encontrado"
Certifique-se de que está na **raiz do projeto**, mesmo nível de `app.py`.

### ❌ Embed falha: "Insufficient quota"
Significa que seus créditos OpenAI acabaram.
Verifique: https://platform.openai.com/account/billing/overview

### ❌ App demora para iniciar
É normal (~2-3s) pois carrega embeddings na primeira execução.
Nas subsequentes é mais rápido.
